# 🧬 Codex 科研技能系统

> 为科研工作者打造的一站式技能调用平台

## ✨ 功能特点

- 🎯 **10项核心技能**：覆盖论文写作、数据分析、文献调研全流程
- ⚡ **智能推荐**：根据研究主题自动推荐适合的技能
- 🔄 **工作流集成**：支持技能组合调用，形成完整的科研流水线
- 📱 **跨平台支持**：支持 Node.js 和浏览器环境

## 🚀 快速开始

```bash
# 安装依赖
npm install

# 运行示例
node examples/basic-usage.js
```
